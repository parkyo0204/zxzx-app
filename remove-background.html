배경 제거

ONNX 모델이 기기에서 배경을 제거합니다. 첫 실행 시 모델 다운로드(수 MB~)가 있어 시간이 걸릴 수 있습니다. 파일당 12MB · 최대 5장.

이미지를 놓거나 클릭해서 선택

브라우저에서 로컬 처리 · 서버 업로드 없음

배경 제거 사용 방법

  1. 도구 열기 — 첫 실행 시 ONNX 모델 다운로드(수 MB~)가 있을 수 있습니다.
  2. 이미지 선택 — 파일당 12MB · 최대 5장. 피사체가 분명한 사진이 유리합니다.
  3. 배경 제거 — 기기에서 추론 후 투명 PNG를 만듭니다.
  4. 다운로드 — 가장자리 확인 후 ZIP/개별 저장. 웹용이면 압축을 이어 하세요.

로컬 배경 제거 개념 도식(데모). 실제 마스크 품질은 원본에 따라 다릅니다.

자주 묻는 질문

서버로 올라가나요?

이미지 데이터는 서버로 가지 않습니다. 모델 파일만 CDN에서 받아 로컬 실행합니다.

출력은?

투명 PNG입니다. 흰 배경 합성이 필요하면 디자인 툴에서 이어가세요.

제한은?

메모리 때문에 파일당 12MB, 최대 5장으로 제한합니다. 큰 이미지는 먼저 리사이즈하세요.

포토샵보다 좋은가요?

단순 컷·속도·프라이버시에는 유리하고, 정밀 마스크는 데스크톱이 우위입니다. 비교 글을 참고하세요.
