최고 성능

배치 처리

완전 무료

숨겨진 비용, 워터마크, 파일 크기 제한 없음. 회원가입 불필요. 영원히 무료.

자주 묻는 질문

이미지가 서버에 업로드되나요?

아닙니다. 모든 이미지 처리가 브라우저에서 JavaScript로 직접 이루어집니다. 파일이 기기를 떠나지 않습니다.

어떤 이미지 포맷을 지원하나요?

JPG, PNG, WebP, AVIF, BMP, GIF, TIFF, SVG를 지원합니다.

파일 크기 제한이 있나요?

파일당 최대 20MB, 배치당 최대 30개 파일까지 처리 가능합니다.

압축이 이미지 품질을 떨어뜨리나요?

스마트 압축 알고리즘이 시각적으로 동일한 품질을 유지하면서 파일 크기를 최대 80% 줄입니다.

모바일에서도 사용할 수 있나요?

네. 모든 최신 모바일 브라우저에서 작동합니다.
