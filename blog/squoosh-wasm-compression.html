Squoosh WASM 코덱으로 타 서비스 대비 월등한 이미지 압축

2026년 7월 15일 · 5분 읽기

대부분의 온라인 이미지 압축 도구는 Canvas API의 기본 인코더를 사용합니다. 이 방식은 간단하지만 압축률과 화질 면에서 한계가 있습니다. ZXZX는 Google Squoosh의 WASM 코덱을 직접 사용하여 이 한계를 극복했습니다.

Squoosh란 무엇인가?

Squoosh는 Google Chrome Labs에서 만든 이미지 압축 도구입니다. GitHub에서 25,000개 이상의 스타를 받았으며, 업계 최고 수준의 이미지 인코더를 브라우저에서 실행할 수 있도록 합니다.

Squoosh의 핵심은 MozJPEG, libwebp, libavif 같은 네이티브 인코더를 WebAssembly(WASM)로 컴파일하여 JavaScript 없이 브라우저에서 직접 실행한다는 것입니다.

왜 WASM이 중요한가?

일반적인 Canvas API 인코더와 WASM 코덱의 차이는 다음과 같습니다:

  • JPG 압축: Canvas 기본 인코더 대비 MozJPEG은 동일 화질에서 10-15% 더 작은 파일 생성
  • WebP 변환: libwebp는 Canvas의 WebP 인코더보다 5-10% 더 효율적
  • AVIF 변환: libavif는 현재 브라우저에 내장된 AVIF 인코더보다 훨씬 뛰어난 품질 제공

ZXZX가 사용하는 기술 스택

  • @jsquash/jpeg — MozJPEG 기반 JPG 인코딩 (압축률 15% 향상)
  • @jsquash/webp — libwebp 기반 WebP 인코딩
  • @jsquash/avif — libavif 기반 AVIF 인코딩 (차세대 포맷)
  • Compressor.js — Canvas 기반 빠른 압축 (폴백)
  • pica — Lanczos3 필터 기반 고품질 리사이즈

어떻게 사용하나?

ZXZX의 이미지 압축기는 특별한 설치 없이 브라우저에서 바로 사용할 수 있습니다. WASM 코덱은 첫 방문 시 자동으로 다운로드되며, 이후에는 캐시되어 즉시 사용 가능합니다.

  1. 압축 도구에 이미지를 드래그 앤 드롭합니다
  2. 품질 수준을 선택합니다 (기본: 80%)
  3. "모두 압축"을 클릭합니다
  4. 압축된 이미지를 개별 또는 ZIP으로 다운로드합니다

결론

ZXZX는 단순한 이미지 압축 도구가 아닙니다. Google Squoosh의 엔진을 직접 활용하여 타 서비스 대비 월등한 압축률과 화질을 제공하면서도, 100% 브라우저 내 처리로 프라이버시를 보장합니다.

무료이고, 회원가입이 필요 없으며, 파일이 서버에 업로드되지 않습니다. 이것이 ZXZX가 다른 이미지 도구와 다른 점입니다.
