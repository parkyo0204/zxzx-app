업스케일

ESRGAN slim 2×/4×. 첫 실행 시 모델 로드가 필요합니다. 파일당 8MB · 최대 3장 · 약 4MP 권장. 실패 시 hqx 폴백.

이미지를 놓거나 클릭해서 선택

브라우저에서 로컬 처리 · 서버 업로드 없음

업스케일 사용 방법

  1. 도구 열기 — 첫 실행 시 ESRGAN 모델 로드가 필요합니다.
  2. 이미지 선택 — 파일당 8MB · 최대 3장 · 약 4MP 권장.
  3. 배율 선택 — 먼저 2×를 권장합니다. 부족할 때만 4×.
  4. 업스케일 후 다운로드 — 실패 시 hqx 폴백. 웹용이면 압축으로 용량을 맞추세요.

자주 묻는 질문

항상 AI가 쓰이나요?

가능하면 ESRGAN을 쓰고, 실패하면 jSquash hqx로 폴백합니다.

제한은?

파일당 8MB, 최대 3장, 약 4MP 권장. 메모리가 부족하면 리사이즈 후 재시도하세요.

업로드되나요?

이미지는 로컬에서만 처리됩니다. 모델은 CDN에서 내려받습니다.

왜 2×가 안전한가요?

4×는 픽셀·메모리·아티팩트가 급증합니다. 자세한 내용은 블로그를 보세요.
